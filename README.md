## 사용 방법
homework_2 해당 폴더에 들어가 $ npm install을 진행하여 필요한 페키지를 다운받습니다.
다운이 완료된후 $ npm start 서버를 실행합니다.

```bash
$ npm install # 디펜던시 인스톨
$ npm start #개발모드 웹 서버 실행
```
